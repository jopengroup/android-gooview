# Android Gooview

一个自定义粘性控件，拖动GooView，在一定的范围内具有粘性效果，当超出范围时，GooView消失。

# Sample usage

A sample project which provides runnable code examples that demonstrate uses of the classes in this project is available in the gooview-app/ folder.
